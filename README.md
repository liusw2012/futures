# C++ Hello World with CMake

A simple C++ "Hello, World!" program built using CMake.

## Build Instructions

1.  Create a build directory:
    ```bash
    mkdir build
    ```
2.  Navigate into the build directory:
    ```bash
    cd build
    ```
3.  Run CMake to configure the project:
    ```bash
    cmake ..
    ```
4.  Compile the project using Make:
    ```bash
    make
    ```

## Running the Executable

After successful compilation, the `helloworld` executable will be located in the `build` directory.

To run the program, navigate to the `build` directory and execute:
```bash
./helloworld
```
This will print "Hello, World!" to your console.
